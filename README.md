# AI Project

Minimal AI project workspace. Contains scripts for generating and evaluating questions, parsing resumes, and running the app.

## Files

- `app.py` — main app entry
- `llm.py` — LLM wrapper
- `question_generator.py` — question generator
- `question_parser.py` — question parser
- `answer_evaluator.py` — answer evaluator
- `resume_parser.py` — resume parsing utilities
- `resume_analyzer.py` — resume analysis
- `requirements.txt` — Python dependencies

## Quick start (Windows PowerShell)

1. Create and activate a virtual environment:

```powershell
python -m venv .venv
. .venv\Scripts\Activate.ps1
```

2. Install dependencies:

```powershell
pip install -r requirements.txt
```

3. Run the app (example):

```powershell
python app.py
```
